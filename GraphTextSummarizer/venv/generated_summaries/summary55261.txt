If we can keep these tendencies and their causes in mind, we're more likely to end up with sensible security measures that are commensurate with the threat, instead of a lot of security theater and draconian police powers that are not. But aside from basic psychology, there are other reasons that it's smart politics to exaggerate terrorist threats, and security threats in general. One of them preaches fear and draconian security measures. This leads to another phenomenon about security and government. If he pushes to relax security, he gets the blame if something bad happens as a result.
