For now, Windows RT and the basic version of Windows 8 are only available pre-installed on new computers. In Windows 8, the iPad has its first real challenger Windows RT and Windows 8 look the same, but there's one key difference: Windows RT will not run your old Windows applications, only applications available through the Windows Store.
